# LP Bradesco NEXT Corretora RJ

Projeto organizado para abrir direto no VS Code.

## Estrutura

- `index.html`: página principal
- `assets/css/style.css`: estilos do site
- `assets/js/script.js`: scripts de menu, carrossel, formulário e WhatsApp
- `assets/img/`: imagens, logos, ícones e banners

## Como visualizar

Abra a pasta no VS Code e use a extensão Live Server.
Também é possível abrir o `index.html` diretamente no navegador.
